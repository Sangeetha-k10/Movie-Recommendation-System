🎬 Movie Recommendation System

A **Machine Learning-based Movie Recommendation System** that suggests movies based on user preferences using content-based filtering techniques. This project helps users discover movies similar to their interests by analyzing movie metadata.

 📌 Project Overview

This system recommends movies by comparing features such as genres, keywords, cast, and crew. It uses **cosine similarity** to find and display movies that are most similar to the selected movie.


⚙️ Features

* Recommend movies based on similarity
* Content-based filtering approach
* Fast and efficient recommendations
* Interactive UI built with Streamlit
* Uses real-world movie dataset

🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Pickle

working

1. Data preprocessing is performed on the movie dataset
2. Important features (genres, cast, crew, keywords) are combined
3. Text data is converted into vectors using vectorization techniques
4. Cosine similarity is applied to measure similarity between movies
5. Top similar movies are recommended to the user

🚀 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Sangeetha-k10/movie-recommendation-system.git

# Navigate to the project folder
cd movie-recommendation-system

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
```
